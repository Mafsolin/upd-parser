Положите сюда фотографии УПД.

Каждый файл обрабатывается как отдельный УПД и добавляется в общий Excel-файл.
Для многостраничного УПД используйте desktop-версию и добавляйте страницы по отдельности только если каждая страница содержит самостоятельные товарные данные.

Поддерживаемые форматы: jpg, jpeg, png, webp, bmp, tiff.
